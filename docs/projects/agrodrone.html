AgroDrone Rover System

ESP32 Arduino Python CNN Unity OpenCV

Overview

Hackathon-winning autonomous agricultural system combining drone and rover navigation with real-time plant disease detection. Built during a 24-hour hackathon at KL University Hyderabad — won first place.

System Components

  • Autonomous Rover: Arduino-based lawn mower pattern navigation for field coverage
  • Drone Simulation: Unity-based drone flight simulation for aerial field scanning
  • CNN Classification Model: Plant disease detection trained on PlantDoc dataset (TensorFlow/Keras)
  • Heat Map Generator: Real-time heat map from drone sensor data for crop health visualization
  • Real-time Disease Detection: Live camera feed processing with OpenCV for on-site diagnosis

Key Features

  • End-to-end agricultural automation from aerial scanning to ground-level treatment
  • CNN model achieving competitive accuracy on PlantDoc benchmark
  • Real-time inference on edge devices for immediate feedback
  • Heat map visualization for systematic crop health monitoring
