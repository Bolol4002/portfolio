Adaptive RTL Control Unit

SystemVerilog Xilinx Vivado Power-Aware Design

Overview

An adaptive control unit implemented at the RTL level that dynamically switches between two control strategies: a power-efficient mode and a high-performance mode. Demonstrates how RTL-level coding decisions significantly influence power and performance without modifying the datapath.

Design Concepts

  • Power-Efficient Control Unit: Minimizes signal switching, uses conditional decoding and gated logic, optimized for low dynamic power consumption
  • High-Performance Control Unit: Uses parallel decoding and precomputed control signals, minimizes critical path delay, higher throughput
  • Adaptive Wrapper: Selects between the two units via a mode signal, enabling dynamic runtime switching

Key Features

  • Two distinct RTL implementations with measurable power/performance trade-offs
  • Runtime mode switching without datapath modification
  • Full Vivado synthesis flow with power, timing, and resource analysis
  • Comparison metrics: dynamic power, LUT utilization, FF count, critical path delay, timing slack

Modules

  • low_power_control_unit.sv — Gated logic, conditional decoding
  • high_perf_control_unit.sv — Parallel decoding, precomputed signals
  • adaptive_control_unit.sv — Mode selection wrapper
  • top_adaptive_control.sv — Top-level integration

Tools

  • HDL: SystemVerilog
  • Synthesis: Xilinx Vivado
  • Simulation: Vivado Simulator
