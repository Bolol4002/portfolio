8-bit ALU Design

Verilog Xilinx Vivado RTL Design

Overview

A synthesizable 8-bit Arithmetic Logic Unit supporting a range of arithmetic and logical operations. Built as a foundational RTL design exercise demonstrating Verilog coding and Vivado synthesis flow.

Supported Operations

  • Arithmetic: ADD, SUB, INC, DEC
  • Logical: AND, OR, XOR, NOT
  • Shift: Left shift, Right shift
  • Comparison: Equal, Greater than, Less than

Key Features

  • Synthesizable RTL with clean Verilog coding
  • Opcode-driven operation selection
  • Status flags: zero, carry, overflow, negative
  • Successfully synthesized on Xilinx Vivado
  • Testbench with exhaustive opcode coverage
