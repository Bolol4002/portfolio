Mini RISC-V Processor

SystemVerilog Xilinx Vivado CPU Design

Overview

A pipelined RISC-V CPU core implementing the 5-stage instruction pipeline. Designed to demonstrate deep knowledge of CPU microarchitecture, pipeline hazards, and digital logic design in SystemVerilog.

Pipeline Stages

  • IF (Instruction Fetch): Fetches instructions from instruction memory using PC
  • ID (Instruction Decode): Decodes instruction, reads register file, generates control signals
  • EX (Execute): ALU operations, branch address calculation, forwarding muxes
  • MEM (Memory): Data memory read/write for load/store instructions
  • WB (Write Back): Writes results back to register file

Key Features

  • Full 5-stage pipeline with hazard detection unit
  • Data forwarding path to resolve data hazards
  • Register file with dual read ports and single write port
  • Support for R-type, I-type, S-type, and B-type instructions
  • Synthesizable on Xilinx Vivado
