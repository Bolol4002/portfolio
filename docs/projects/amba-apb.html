AMBA APB Memory Interface

SystemVerilog AMBA Protocol Verification

Overview

A protocol-compliant AMBA APB (Advanced Peripheral Bus) memory interface implemented from scratch in SystemVerilog. Includes a constrained-random testbench achieving high functional coverage for design correctness validation.

Protocol Details

  • Setup Phase: Address, write data, and control signals driven by master
  • Access Phase: Slave responds with read data or latches write data
  • Handshake: PSEL, PENABLE, PREADY信号 coordination
  • Transfer Types: Read and write transactions with proper wait state handling

Key Features

  • Full APB protocol compliance with all transfer phases
  • Constrained-random testbench for stimulus generation
  • Functional coverage model tracking all protocol states
  • Self-checking testbench with automatic result reporting
