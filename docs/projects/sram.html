SRAM Design & Verification

SystemVerilog SVA Verification

Overview

A synchronous SRAM module designed and verified using modern verification methodologies. Features a constrained-random testbench with SystemVerilog Assertions (SVA) for rigorous functional validation.

Design

  • Synchronous read/write operations synchronized to clock edge
  • Parameterized address and data bus widths
  • Byte-enable write mask support
  • Single-port configuration

Verification

  • Constrained-random address and data generation
  • SVA assertions checking read-after-write consistency
  • Data integrity verification across all memory locations
  • Timing checks for setup and hold violations
  • Coverage-driven verification for exhaustive stimulus

Key Features

  • Modern verification: constrained-random + assertions + coverage
  • Self-checking testbench with automatic pass/fail reporting
  • Reusable verification components
