Drowsiness Detection System

ESP32-CAM OpenCV Python Embedded

Overview

A real-time driver fatigue detection system combining embedded hardware (ESP32-CAM) with computer vision (OpenCV) to monitor eye closure patterns and trigger alerts when drowsiness is detected.

System Architecture

  • Hardware: ESP32-CAM module captures live video feed
  • Processing: Python backend with OpenCV for face and eye detection
  • Detection: Eye Aspect Ratio (EAR) computation for blink tracking
  • Alert: Audio/visual alarm triggered on sustained eye closure

Key Features

  • Real-time video processing with low latency
  • EAR-based drowsiness metric for reliable detection
  • Embedded + software integration for portable deployment
  • Configurable sensitivity thresholds
