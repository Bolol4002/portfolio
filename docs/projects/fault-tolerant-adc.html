Fault-Tolerant ADC

QSPICE Analog Design IoT

Overview

A fault-tolerant analog-to-digital converter designed for edge IoT devices. Built entirely in QSPICE, this project implements a complete ADC pipeline with an added fault detection and protection block that monitors supply voltage integrity.

Architecture

  • Comparator: Ideal comparator subcircuit comparing analog input against reference voltage
  • Priority Encoder: 8-to-3 priority encoder converting comparator outputs to 3-bit digital code
  • Fault Detection Block: Monitors Vref (7V–9V safe range), raises error flag and blocks signal on out-of-range conditions
  • Resistor Ladder: Generates reference voltage levels for the comparators

Key Features

  • Complete ADC pipeline: analog input → comparators → priority encoder → digital output
  • Fault detection with automatic error flagging and signal isolation
  • Safe voltage range monitoring (7V–9V) for IoT edge reliability
  • Subcircuit-based modular design for reusability
